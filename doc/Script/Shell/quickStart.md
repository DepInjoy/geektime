## `case esac`语句

`case ... esac` 与其他语言中的`switch ... case`语句类似，是一种多分枝选择结构。case 语句匹配一个值或一个模式，如果匹配成功，执行相匹配的命令。case 语句格式如下：

```shell
case 值 in
模式1)
    command1
    ;;
模式2）
    command2
    ;;
*)
    command3
    ;;
esac
```

匹配发现取值符合某一模式后，其间所有命令开始执行直至`;;`，`;;` 与其他语言中的`break`类似。如果无一匹配模式，使用星号`*`捕获该值，再执行后面的命令。
